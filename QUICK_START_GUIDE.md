## 📋 Daily Workflow

### **1. Start of Day**
```
□ Log into admin dashboard
□ Check for overnight orders
□ Review sold-out items from yesterday
□ Update menu availability if needed
```

### **2. During Service**
```
□ Keep admin dashboard open
□ Watch for new orders (appear automatically)
□ Print kitchen ticket for each order
□ Update status: Received → Preparing → Ready → Completed
```

### **3. End of Day**
```
□ Review today's revenue and order count
□ Mark any remaining orders as completed
□ Note any menu items to restock
```

## 🎯 Common Tasks

### **Managing New Orders**
1. Order appears in dashboard automatically
2. Click "Details" to expand order
3. Review items and special instructions
4. Click 🖨️ Print icon to print kitchen ticket
5. Update status as you progress

### **Marking Items Sold Out**
1. Go to **Menu Management** (/admin/menu)
2. Find the item (use search if needed)
3. Click **"Mark Sold Out"** button
4. Customers immediately see "SOLD OUT" badge
5. Click again to unmark when restocked

### **Updating Prices**
1. Go to **Menu Management** (/admin/menu)
2. Click **"Edit"** next to item price
3. Enter new price
4. Click **"Save"**
5. Price updates immediately for all customers

### **Printing Kitchen Tickets**
1. In dashboard, find the order
2. Click the 🖨️ **Printer icon**
3. Print window opens automatically
4. Print to kitchen printer or save as PDF

## 📊 Order Status Guide

**Update order status by clicking the status buttons:**

| Status | When to Use |
|--------|-------------|
| **Received** | Order just came in (automatic) |
| **Preparing** | Kitchen has started working on it |
| **Ready** | Order is complete and ready for pickup/delivery |
| **Out for Delivery** | Driver has left with order (delivery only) |
| **Completed** | Customer picked up or received delivery |

| Problem | Solution |
