# Images Folder

## Required Images:

1. **hilltop-hero-image.png** - Hero section background image
   - Recommended size: 1920x1080px or larger
   - Should show the restaurant interior/exterior or food
   
2. **hilltop-logo.png** - Restaurant logo
   - Recommended size: 200x60px (or similar proportions)
   - Transparent background preferred
   - Will appear in the navbar

## How to Add:

Simply copy your images to this folder:
- /Users/dylanseis/Desktop/ordering-app/public/images/

The images will be automatically served at:
- /images/hilltop-hero-image.png
- /images/hilltop-logo.png
