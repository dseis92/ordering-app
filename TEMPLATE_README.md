## 🚀 Quick Start (For Developers)

### **Option 1: New Project from Scratch**

1. **Clone the Hilltop Ordering App**
   ```bash
   git clone [your-template-repo-url]
   cd ordering-app
   npm install
   ```

2. **Customize for New Client**
   ```bash
   # Copy config template
   cp CLIENT_CONFIG_TEMPLATE.js src/brand.config.js

   # Edit with client info (search for [CLIENT_* placeholders)
   code src/brand.config.js

   # Update menu data
   code src/data/menu.js

   # Update colors
   code tailwind.config.js
   ```

3. **Test Locally**
   ```bash
   npm run dev
   # Visit http://localhost:5173
   ```

4. **Deploy to Production**
   ```bash
   # Follow IMPLEMENTATION_CHECKLIST.md Day 15-16
   vercel --prod
   ```

### **Option 2: Use Existing Hilltop App as Template**

1. **Fork the Repository**
2. **Find and Replace**
   - Search: "Hilltop Pub and Grill" → Replace: "[New Business Name]"
   - Search: "hilltop-green" → Replace: "[new-brand-color]"
   - Update all references in:
     - `src/brand.config.js`
     - `src/data/menu.js`
     - `tailwind.config.js`
     - `package.json`

3. **Follow Implementation Checklist**
   - Use `IMPLEMENTATION_CHECKLIST.md` as your guide
   - 28-day timeline from start to launch

```

Cons: More complex tracking, variable income
```

## 🛠️ Customization Guide

### **Must Customize for Every Client:**
1. **Business Info** (`src/brand.config.js`)
   - Name, address, phone, hours
   - Tax rate, delivery fee, timing estimates

2. **Menu Data** (`src/data/menu.js`)
   - All menu items with prices
   - Categories
   - Customization options

3. **Branding** (`tailwind.config.js` + images)
   - Logo (`/public/images/logo.png`)
   - Hero image (`/public/images/hero-image.png`)
   - Brand colors
   - Fonts (if needed)

4. **Database** (`supabase-schema.sql`)
   - Run in client's Supabase account
   - Configure RLS policies

5. **Deployment** (Vercel)
   - Deploy to client's account or yours
   - Set environment variables
   - Configure custom domain (optional)

### **Optional Customizations:**
- Payment integration (Stripe)
- Email notifications (Resend)
- SMS notifications (Twilio)
- Custom features per client needs
- Advanced analytics
- Multi-location support
